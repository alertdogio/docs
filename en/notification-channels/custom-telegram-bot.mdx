## Setup Steps

<Steps>

<Step title="Create Group and Get Chat ID">
  Open Telegram, create a new group (or Channel)

  <Frame caption="Create a Telegram Group">
  <img
    src="/images/notification-channels/telegram/telegram-create-group.png"
    alt="Create Group"
  />
</Frame>

Search for MCT_CLUB_BOT and add it to the group

  <Frame caption="Search MCT_CLUB_BOT">
  <img
    src="/images/notification-channels/telegram/telegram-search-mct-bot.png"
    alt="Telegram search MCT_CLUB_BOT"
  />
</Frame>

 <Frame caption="Add MCT_CLUB_BOT to Group">
  <img
    src="/images/notification-channels/telegram/telegram-add-mct-bot.png"
    alt="Add MCT Bot to Telegram group"
  />
</Frame>
After successfully added, MCT Bot will automatically send the Chat ID in the group, copy the Chat ID for later use

  <Frame caption="Telegram Group Chat ID">
  <img
    src="/images/notification-channels/telegram/telegram-copy-chat-id.png"
    alt="Telegram group Chat ID"
  />
</Frame>

</Step>



<Step title="Create Your Bot and Get API Token">
Click [@BotFather](https://t.me/BotFather) link to automatically open Telegram and chat with BotFather, click **<Icon icon="list" />menu** and select **/newbot**
   <Frame caption="Create New Bot">
  <img
    src="/images/notification-channels/telegram/telegram-bot-father.png"
    alt="Create new bot and give it a unique name"
  />
</Frame>


Set an English username for the Bot (username cannot duplicate with others' bots), and must end with `bot`
   <Frame caption="Screenshot of Setting Bot English Name">
  <img
    src="/images/notification-channels/telegram/telegram-set-bot-username.png"
    alt="Set an unused English name for Bot"
  />
</Frame>

After correctly setting the username, you will get the API Token, copy it for the next step
   <Frame caption="Copy API">
  <img
    src="/images/notification-channels/telegram/telegram-copy-api-token.png"
    alt="Copy API"
  />
</Frame>

</Step>

<Step title="Invite Custom Bot to Join Group">
Add the custom Telegram Bot created in the previous step to the group with @MCT_CLUB_BOT created in the first step
<Frame caption="Click +ADD to Add Group Member">
  <img
    src="/images/notification-channels/telegram/telegram-invite-custom-bot.png"
    alt="Telegram add new member to group"
  />
</Frame>

Enter your custom Bot name
<Frame caption="Search and Invite Custom Bot to Group">
  <img
    src="/images/notification-channels/telegram/telegram-custom-bot-added.png"
    alt="Search and invite custom Bot to group"
  />
</Frame>
</Step>

<Step title="Create Custom Bot Notification Channel">
Open AlertDog **Add New Channel** -> **Telegram** -> **Notify to Group** -> **Custom BOT**, enter the Chat ID and API Token from the previous two steps -> **Connect** to confirm
<Frame >
  <img
    src="/images/notification-channels/telegram/telegram-custom-bot-created.png"
    alt="Create custom Telegram Bot notification channel in AlertDog"
  />
</Frame>
<Check>Wow! You've mastered such a challenging operation, impressive! You have successfully completed the Telegram custom bot binding</Check>
</Step>


</Steps>



## FAQ

<AccordionGroup>
<Accordion title="Unable to set up custom Bot?">
  Custom Bot requires VIP upgrade to use: https://alertdog.io/price
</Accordion>

<Accordion title="Custom Bot username setup unsuccessful">
  Username setup must satisfy both of the following: 1. Username cannot be the same as others; 2. Must end with 'bot' and avoid using numbers
</Accordion>

<Accordion title="Can I create and bind multiple custom Bots?">
  Yes.
</Accordion>
</AccordionGroup>

## Add Rune Minting Monitoring

Precisely capture trending rune minting opportunities by setting conditions based on time and quantity dimensions.


<Frame caption="Rune Minting Monitoring Configuration">
  <img

  />

**Condition Setting Description**

<Note>
**Condition Setting: Configure through time and quantity dimensions**

**Time Dimension Options:**

- Within last 5 minutes mempool
- Within last 10 minutes mempool
- Last 3 blocks
- Last 10 blocks
- Last 20 blocks

**Quantity Dimension Options:**

- Minting wallet count greater than
- Transaction count greater than
- Rune minting progress greater than
- Rune minting progress less than

You can customize combinations of time conditions + quantity conditions. Multiple conditions can be added, notifications will only be sent when runes meet all conditions simultaneously.

## Manage Rune Monitoring

You can manage added rune monitoring through the following methods:
- **Filter Monitoring**: Filter specific monitoring by monitoring type
- **Batch Operations**: Click batch operations to perform batch move to group, batch edit, batch modify notification methods, batch delete on selected monitoring
- **Monitoring History**: View received monitoring notification history on the unified monitoring history page
- **Enable/Disable, Edit, Delete**: Perform enable/disable, edit, delete operations on monitoring

When you encounter a 429 error code while using the monitoring platform, it means your request has triggered system limits. This is usually caused by the following reasons, which can be investigated in order:



## Situation 1: Too Many Requests in Short Time

If you send a large number of requests in a very short time, it may trigger the 429 rate limit protection mechanism.

<Tip>Solution: Try logging out and logging back in, and be careful not to refresh the page frequently</Tip>


## Situation 2: Computer System Clock Out of Sync

Check if your computer time has a discrepancy with the standard time.
<Tip>Solution: Go to your computer system settings to synchronize time</Tip>

## Situation 3: Free Version Request Frequency Limit

Free version users have daily request limits. When the limit is reached, you'll receive a 429 error.

<Tip>Solution: [Upgrade to Pro](https://alertdog.io/pricing?utm_source=docs_faq_en) to unlock more features</Tip>

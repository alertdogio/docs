Introduction: FeiShu is an enterprise-level collaboration platform that supports pushing monitoring messages through group robots, suitable for team use, allowing real-time viewing of monitoring alerts in FeiShu groups.

Usage Instructions: After configuration, you can select FeiShu as the notification method when creating monitoring tasks. When monitoring triggers, messages will be pushed to the specified FeiShu group.

## FAQ

<AccordionGroup>
<Accordion title="How to add a robot in FeiShu?">
  In FeiShu group settings → Group Robot → Add Robot → Custom Robot to create and get the Webhook URL.
</Accordion>

<Accordion title="Can I add multiple FeiShu groups?">
  Yes. You can configure multiple Webhooks for different FeiShu groups to implement categorized notifications.
</Accordion>

<Accordion title="Does FeiShu robot have message frequency limits?">
  FeiShu robots have certain frequency limits. Please refer to FeiShu's official documentation for specific limits.
</Accordion>
</AccordionGroup>

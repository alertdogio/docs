Introduction: FeiShu is an enterprise-level collaboration platform that supports pushing monitoring messages through group bots, suitable for teams, allowing real-time viewing of monitoring alerts in FeiShu groups.

## Step 3: Create Notification Channel
  Open AlertDog, select FeiShu, enter the secret and Webhook URL, then confirm
   <Frame caption="Create Notification Channel">
  <img
    src="/images/notification-channels/dingtalk/dingtalk-alertdog-config.png"
    alt="Create notification channel"
  />
</Frame>

<Note>
  You need to add a custom bot in the FeiShu group, then copy the bot's Webhook URL to AlertDog.
</Note>

Usage Instructions: After configuration, you can select FeiShu as the notification method when creating monitoring tasks. When monitoring is triggered, messages will be pushed to the specified FeiShu group.

## FAQ

<AccordionGroup>
<Accordion title="How to add a bot in FeiShu?">
  In FeiShu group settings → Group Bots → Add Bot → Create a custom bot and get the Webhook URL.
</Accordion>

<Accordion title="Can I add multiple FeiShu groups?">
  Yes. You can configure multiple Webhooks for different FeiShu groups to implement categorized notifications.
</Accordion>

<Accordion title="Do FeiShu bots have message frequency limits?">
  FeiShu bots have certain frequency limits, please refer to FeiShu's official documentation for specific limits.
</Accordion>
</AccordionGroup>

自定义 Bot 需升级 Pro 后使用
</Note> 


## 第一步：创建群组并获取 Chat ID
  打开 Telegram，创建一个新的群组（或Channel）

  <Frame caption="创建一个电报群组">
  <img
    src="/images/notification-channels/telegram/telegram-create-group.png"
    alt="创建群组"
  />
</Frame>

搜索 MCT_CLUB_BOT 并将其添加到群组

  <Frame caption="搜索 MCT_CLUB_BOT">
  <img
    src="/images/notification-channels/telegram/telegram-search-mct-bot.png"
    alt="Telegram 搜索 MCT_CLUB_BOT"
  />
</Frame>

 <Frame caption="将 MCT_CLUB_BOT 拉入群组">
  <img
    src="/images/notification-channels/telegram/telegram-add-mct-bot.png"
    alt="Telegram 添加 MCT Bot 到群组"
  />
</Frame>
添加成功后，MCT Bot 将会自动在群组里发送 Chat ID，复制 Chat ID 备用

  <Frame caption="Telegram 群组的 Chat ID">
  <img
    src="/images/notification-channels/telegram/telegram-copy-chat-id.png"
    alt="Telegram 群组的 Chat ID"
  />
</Frame>


## 第二步：创建你的 Bot 获取 API Token
点 [@BotFather](https://t.me/BotFather) 链接会自动打开 Telegram 与 BotFather 对话，点击 **<Icon icon="list" />菜单** 选择 **/newbot**
   <Frame caption="创建新 Bot">
  <img
    src="/images/notification-channels/telegram/telegram-bot-father.png"
    alt="创建新 Bot 并取一个独一无二的名字"
  />
</Frame>


给 Bot 设置一个英文用户名（用户名不能与他人的bot重复），并且必须以 `bot` 结尾
   <Frame caption="为Bot设置英文名的截图">
  <img
    src="/images/notification-channels/telegram/telegram-set-bot-username.png"
    alt="为Bot设置一个未被使用过的英文名"
  />
</Frame>

正确设置用户名后，将会得到 API Token，复制供下一步使用
   <Frame caption="复制 API">
  <img
    src="/images/notification-channels/telegram/telegram-copy-api-token.png"
    alt="复制 API"
  />
</Frame>


## 第三步：邀请自定义 Bot 加入群组
将上一步创建的自定义 Telegram Bot 拉到我们第一步创建的有 @MCT_CLUB_BOT 的群组
<Frame caption="点击 +ADD 添加组成员">
  <img
    src="/images/notification-channels/telegram/telegram-invite-custom-bot.png"
    alt="Telegram 添加新成员到组"
  />
</Frame>

输入你的自定义 Bot 名称
<Frame caption="搜索并邀请自定义 Bot 到群组">
  <img
    src="/images/notification-channels/telegram/telegram-custom-bot-added.png"
    alt="搜索并邀请自定义 Bot 到群组"
  />
</Frame>

## 第四步：创建自定义 Bot 通知渠道
打开监控狗 **添加新渠道** -> **Telegram** -> **通知到群组** -> **自定义 BOT** ，分别输入前两步的 Chat ID 和 API Token -> **连接** 确认即可
<Frame >
  <img
    src="/images/notification-channels/telegram/telegram-custom-bot-created.png"
    alt="AlertDog 中创建自定义 Telegram Bot 通知渠道"
  />
</Frame>
<Check>哇！这么高难度的操作你都搞定了, 佩服佩服！你已成功完成 Telegram 电报自定义机器人的绑定</Check>

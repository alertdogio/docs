## Operation Steps

<Steps>
<Step title="Open Message Push in WeCom">
  Open WeChat Work, select any group chat, open settings
    <Frame>
  <img
    src="/images/notification-channels/wx/workwechat-robot-settings.png"
    alt="WeChat Work group chat settings interface showing group robot options"
  />
</Frame>
Select message push
  <Frame>
  <img
    src="/images/notification-channels/wx/workwechat-add-robot.png"
    alt="WeChat Work message push interface, click to add robot"
  />
</Frame>
Click add
  <Frame>
  <img
    src="/images/notification-channels/wx/workwechat-robot-name.png"
    alt="WeChat Work customize robot name interface"
  />
</Frame>
Customize a name, click save
  <Frame>
  <img
    src="/images/notification-channels/wx/workwechat-copy-webhook.png"
    alt="WeChat Work copy Webhook URL interface"
  />
</Frame>
Copy the Webhook address, click complete
  <Frame>
  <img
    src="/images/notification-channels/wx/workwechat-alertdog-config.png"
    alt="Configure WeChat Work notification channel in AlertDog"
  />
</Frame>




</Step>

<Step title="Create Notification Channel">
  Open AlertDog, select WeChat Work, enter the Webhook and click connect
  <Frame>
  <img
    src="/images/notification-channels/wx/workwechat-test-notification.png"
    alt="WeChat Work test notification success interface"
  />
</Frame>
</Step>

<Step title="Creation Complete">
After successful connection, the notification channel has been successfully added and will appear in the notification channel list


</Step>
</Steps>

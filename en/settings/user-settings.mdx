On the user settings page, you can view personal account information, configure general settings, view notification counts/feature usage quotas, and subscription records.


### Change Settings

Manage your basic personal information:

<Frame caption="User Settings Page">
  <img
    src="/images/settings/user-settings-overview.png"
    alt="User settings page showing system display, token blacklist, notification language and timezone settings"
  />
</Frame>
- **1. System Display**: Set system color scheme here (dark/light), default is light
- **2. EVM Token Blacklist**: Add by selecting chain and entering token contract address, tokens in the EVM token blacklist will not trigger notifications
- **3. Notification Language**: Select notification language (Simplified Chinese/Traditional Chinese/English), default is Simplified Chinese
- **4. Timezone Settings**: Select timezone, default is UTC+8

## Security Settings

### Password Management

<Note>
  If you sign in using a Web3 wallet or third-party account (such as Google), you do not need to set a password.
</Note>

### API Key Management

API keys are used for programmatic access to AlertDog services:

- **Create API Key**: Generate a new API key for development
- **View Keys**: View list of created API keys
- **Delete Keys**: Delete API keys that are no longer in use

<Warning>
  API keys have full account permissions, please keep them secure and do not share them with others.
</Warning>

## Subscription Management

### Plan Information

View current subscription plan information:

- **Plan Type**: Free, Professional or Enterprise
- **Validity Period**: Plan expiration time
- **Quota Usage**: Number of monitoring tasks, API call counts, etc.

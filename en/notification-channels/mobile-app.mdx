# Setup Steps


## Step 1: Download APP
 Download and install [MCT Official APP](https://mct.xyz/apps)

## Step 2: Log in with Wallet Address
  Log in to MCT Official APP using the same wallet address as the website

## Step 3: Send Test Message
Click the send test button. After receiving the test notification, the notification channel has been successfully added

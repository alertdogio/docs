Introduction: Discord is an ideal notification channel for team collaboration, supporting pushing monitoring messages to specified Discord channels, making it convenient for team members to share monitoring information.

    alt="Discord server settings entry"

</Frame>

<Frame caption="Create New Webhook">
  <img

Copy the Webhook URL.

<Frame caption="Copy Webhook URL">
  <img
    src="/images/notification-channels/discord/discord-copy-webhook-url.png"
    alt="Copy Discord Webhook URL"
  />
</Frame>
</Step>

<Step title="Create Notification Channel">
Open AlertDog, select Discord, enter the Webhook URL and click connect.

<Frame caption="Create Notification Channel">
  <img
    src="/images/notification-channels/discord/discord-alertdog-config.png"
    alt="AlertDog add Discord notification channel, enter Webhook URL"
  />
</Frame>
</Step>

<Step title="Creation Complete">
After successful connection, the notification channel has been successfully added and will appear in the notification channel list.

<Frame caption="Creation Complete">
  <img
    src="/images/notification-channels/discord/discord-channel-created.png"
    alt="Discord notification channel created successfully, shown in list"
  />
</Frame>

</Step>

</Steps>

Usage Instructions: After configuration, you can select Discord as the notification method when creating monitoring tasks. When monitoring triggers, messages will be pushed to the specified Discord channel.

## FAQ

<AccordionGroup>
<Accordion title="How to delete a Discord notification channel?">
  Find the Discord item in the notification channel list and click "Delete".
</Accordion>

<Accordion title="Can I add multiple Discord channels?">
  Yes. You can configure multiple Webhooks for different Discord channels to implement categorized notifications.
</Accordion>

<Accordion title="Where to get the Webhook URL?">
  In Discord server's channel settings → Integrations → Webhook to create and copy the URL.
</Accordion>
</AccordionGroup>

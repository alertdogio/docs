Introduction: DingTalk is an enterprise-level office platform that supports pushing monitoring messages through group robots, suitable for enterprise teams, allowing real-time viewing of monitoring alerts in DingTalk groups.

Usage Instructions: After configuration, you can select DingTalk as the notification method when creating monitoring tasks. When monitoring triggers, messages will be pushed to the specified DingTalk group.

## FAQ

<AccordionGroup>
<Accordion title="How to add a robot in DingTalk?">
  In DingTalk group settings → Smart Group Assistant → Add Robot → Custom Robot to create and get the Webhook URL.
</Accordion>

<Accordion title="What is signature verification?">
  DingTalk robots support signature verification to enhance security. If signature verification is enabled, you need to configure the signature secret in AlertDog.
</Accordion>

<Accordion title="Does DingTalk robot have message frequency limits?">
  DingTalk robots have certain frequency limits, a maximum of 20 messages per minute.
</Accordion>
</AccordionGroup>

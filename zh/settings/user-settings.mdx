在用户设置页面，您可以查看个人账户信息、进行通用设置、查看通知条数/各功能权限用量及订购记录。


### 更改设置

管理您的基本个人信息：

<Frame caption="用户设置页面">
  <img
    src="/images/settings/user-settings-overview.png"
    alt="用户设置页面，显示系统显示、代币黑名单、通知语言和时区设置"
  />
</Frame>
- **1、系统显示**：在这里设置系统颜色（深色/浅色），默认浅色
- **2、EVM代币黑名单**：通过选择链和输入代币合约地址进行添加，涉及EVM代币黑名单的代币不会进行通知
- **3、通知语言**：选择通知语言（简体中文/繁体中文/英文），默认简体中文
- **4、时区设置**：选择时区，默认东八区

## 安全设置

### 密码管理

<Note>
  如果您使用 Web3 钱包或第三方账号（如 Google）登录，则无需设置密码。
</Note>

### API 密钥管理

API 密钥用于程序化访问 AlertDog 服务：

- **创建 API 密钥**：生成新的 API 密钥用于开发
- **查看密钥**：查看已创建的 API 密钥列表
- **删除密钥**：删除不再使用的 API 密钥

<Warning>
  API 密钥具有完整的账户权限，请妥善保管，不要泄露给他人。
</Warning>

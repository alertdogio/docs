## 添加地址监控

BTC 地址监控支持监控多种资产类型，包括 BTC 转账、符文（Runes）和铭文（Inscriptions）。

| 字段         | 说明                                                        | 备注                                                                 |
| ------------ | ----------------------------------------------------------- | -------------------------------------------------------------------- |
| **钱包地址** | 您要监控的钱包地址                                               | 需是有效的 BTC 地址                                          |
| **钱包名称** | 钱包地址的备注名称，方便记忆和管理                                    | 同一个钱包组内不能有相同的钱包名称                                   |
| **钱包组**   | 将监控归属于组内，方便后续查看和管理                                   | 首次使用需要创建钱包组，钱包组名不能重复                             |
| **监控类型** | **BTC**：监控 BTC 转账<br />**符文**：监控符文交易<br />**铭文**：监控铭文转移/接收/买卖 | 可以选择多种类型同时监控                                         |
| **通知方式** | 监控通知消息将会通过该方式发送                                     | 可以同时选择多个通知方式；<br />如果选择的通知方式均已停用，该监控自动停用 |
| **通知保留** | 监控通知历史在 AlertDog 网页保留的时间                             | 默认是 1 天，超过时间自动清除                                        |

## 管理地址监控

您可以通过以下方式管理已添加的地址监控：

- **筛选钱包组管理**：在选择指定的钱包组后，可以对钱包组的名称进行修改或者删除钱包组
- **批量操作**：点击批量操作，可以对选中的钱包进行批量移动组、批量编辑、批量修改通知方式、批量删除的操作
- **导出**：钱包列表数据支持导出 Excel 格式的表格
- **监控历史**：可以在统一的监控历史页查看已收到的监控通知历史
- **搜索**：可以搜索指定的钱包
- **启用/停用、编辑、删除**：可以对钱包进行启用/停用、编辑、删除的操作

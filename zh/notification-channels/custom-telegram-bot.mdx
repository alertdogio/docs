自定义 Bot 需升级 Pro 后使用
</Note> 



## 操作步骤

<Steps>

<Step title="获取 Chat ID">
  打开 Telegram，创建群组并邀请 MCT Bot

  <Frame caption="创建群组">
  <img
    src="/images/notification-channels/telegram/telegram-create-group.png"
    alt="创建群组"
  />
</Frame>

搜索 MCT_CLUB_BOT 并将其添加到群组


  <Frame >
  <img
    src="/images/notification-channels/telegram/telegram-search-mct-bot.png"
    alt="Telegram 搜索 MCT_CLUB_BOT"
  />
</Frame>

 <Frame >
  <img
    src="/images/notification-channels/telegram/telegram-add-mct-bot.png"
    alt="Telegram 添加 MCT Bot 到群组"
  />
</Frame>
添加成功后，MCT Bot 将会自动在群组里发送 Chat ID，复制 Chat ID

  <Frame caption="复制 Chat ID">
  <img
    src="/images/notification-channels/telegram/telegram-copy-chat-id.png"
    alt="复制 Chat ID"
  />
</Frame>

</Step>



<Step title="获取 Token">
在 Telegram 上访问 https://t.me/BotFather，点击菜单按钮并选择 /newbot
   <Frame caption="创建新 Bot">
  <img
    src="/images/notification-channels/telegram/telegram-bot-father.png"
    alt="创建newbot"
  />
</Frame>

自定义一个 Bot 名称
   <Frame caption="自定义名称">
  <img
    src="/images/notification-channels/telegram/telegram-newbot-command.png"
    alt="自定义名称"
  />
</Frame>

给 Bot 设置一个用户名（用户名不能与他人相同），并且必须以 'bot' 结尾
   <Frame caption="设置用户名">
  <img
    src="/images/notification-channels/telegram/telegram-set-bot-username.png"
    alt="设置用户名"
  />
</Frame>

获取 API Token，设置完成用户名后，复制 API
   <Frame caption="复制 API">
  <img
    src="/images/notification-channels/telegram/telegram-copy-api-token.png"
    alt="复制 API"
  />
</Frame>

</Step>

<Step title="将自定义 Bot 邀请到群组">
将自定义 Telegram Bot 邀请到有 @MCT_CLUB_BOT 的群组
<Frame >
  <img
    src="/images/notification-channels/telegram/telegram-invite-custom-bot.png"
    alt="Telegram 搜索并邀请自定义 Bot 到群组"
  />
</Frame>
<Frame >
  <img
    src="/images/notification-channels/telegram/telegram-custom-bot-added.png"
    alt="Telegram 自定义 Bot 添加到群组成功"
  />
</Frame>
</Step>

<Step title="创建通知渠道">
打开 AlertDog，选择 Telegram，选择自定义 Bot，分别输入 Chat ID 和 API，确认即可
<Frame >
  <img
    src="/images/notification-channels/telegram/telegram-custom-bot-created.png"
    alt="AlertDog 中创建自定义 Telegram Bot 通知渠道"
  />
</Frame>

</Step>


</Steps>



## 常见问题

<AccordionGroup>
<Accordion title="无法设置自定义 Bot？">
  自定义 Bot 需升级 VIP 后使用：https://alertdog.io/price
</Accordion>

<Accordion title="自定义 Bot 用户名设置不成功">
  用户名设置注意需同时满足以下两点：1. 用户名不能与他人相同；2. 必须以 'bot' 结尾，并且不要使用数字
</Accordion>

<Accordion title="可以创建绑定多个自定义 Bot 吗？">
  可以。
</Accordion>
</AccordionGroup>

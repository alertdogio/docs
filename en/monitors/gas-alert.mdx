## Manage GAS Monitoring

You can manage added GAS monitoring through the following methods:

- **Filter Monitoring**: Filter specific monitoring by monitoring type
- **Batch Operations**: Click batch operations to perform batch move to group, batch edit, batch modify notification methods, batch delete on selected monitoring
- **Monitoring History**: View received monitoring notification history on the unified monitoring history page
- **Enable/Disable, Edit, Delete**: Perform enable/disable, edit, delete operations on monitoring

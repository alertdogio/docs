## Operation Steps

<Steps>

<Step title="Get Chat ID">
  Open Telegram, create a group and invite MCT Bot

  <Frame caption="Create Group">
  <img
    src="/images/notification-channels/telegram/telegram-create-group.png"
    alt="Create group"
  />
</Frame>

Search for MCT_CLUB_BOT and add it to the group


  <Frame >
  <img
    src="/images/notification-channels/telegram/telegram-search-mct-bot.png"
    alt="Telegram search for MCT_CLUB_BOT"
  />
</Frame>

 <Frame >
  <img
    src="/images/notification-channels/telegram/telegram-add-mct-bot.png"
    alt="Telegram add MCT Bot to group"
  />
</Frame>
After successful addition, MCT Bot will automatically send the Chat ID in the group, copy the Chat ID

  <Frame caption="Copy Chat ID">
  <img
    src="/images/notification-channels/telegram/telegram-copy-chat-id.png"
    alt="Copy Chat ID"
  />
</Frame>

</Step>



<Step title="Get Token">
Visit https://t.me/BotFather on Telegram, click the menu button and select /newbot
   <Frame caption="Create New Bot">
  <img
    src="/images/notification-channels/telegram/telegram-bot-father.png"
    alt="Create newbot"
  />
</Frame>

Customize a Bot name
   <Frame caption="Customize Name">
  <img
    src="/images/notification-channels/telegram/telegram-newbot-command.png"
    alt="Customize name"
  />
</Frame>

Set a username for the Bot (username cannot be the same as others), and must end with 'bot'
   <Frame caption="Set Username">
  <img
    src="/images/notification-channels/telegram/telegram-set-bot-username.png"
    alt="Set username"
  />
</Frame>

Get API Token, after setting the username, copy the API
   <Frame caption="Copy API">
  <img
    src="/images/notification-channels/telegram/telegram-copy-api-token.png"
    alt="Copy API"
  />
</Frame>

</Step>

<Step title="Invite Custom Bot to Group">
Invite the custom Telegram Bot to the group with @MCT_CLUB_BOT
<Frame >
  <img
    src="/images/notification-channels/telegram/telegram-invite-custom-bot.png"
    alt="Telegram search and invite custom Bot to group"
  />
</Frame>
<Frame >
  <img
    src="/images/notification-channels/telegram/telegram-custom-bot-added.png"
    alt="Telegram custom Bot successfully added to group"
  />
</Frame>
</Step>

<Step title="Create Notification Channel">
Open AlertDog, select Telegram, select custom Bot, enter Chat ID and API respectively, confirm
<Frame >
  <img
    src="/images/notification-channels/telegram/telegram-custom-bot-created.png"
    alt="Create custom Telegram Bot notification channel in AlertDog"
  />
</Frame>

</Step>


</Steps>



## FAQ

<AccordionGroup>
<Accordion title="Cannot set custom Bot?">
  Custom Bot requires VIP upgrade to use: https://alertdog.io/pricing?utm_source=docs_en
</Accordion>

<Accordion title="Custom Bot username setup unsuccessful">
  When setting the username, note that it must meet both of the following: 1. Username cannot be the same as others; 2. Must end with 'bot', and do not use numbers
</Accordion>

<Accordion title="Can I create and bind multiple custom Bots?">
  Yes.
</Accordion>
</AccordionGroup>
